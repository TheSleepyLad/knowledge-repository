# 白名单审核评分标准

更新时间：2026-07-07

用途：用于审核视频博主、机构账号、官方平台、课程来源是否可以进入正式采集白名单。你和 Codex 各自收集的候选账号，最后都按此标准评分后合并。

## 审核结论

| 结论 | 条件 | 后续动作 |
|---|---|---|
| A 正式白名单 | 总分 >= 85，且无一票否决项 | 可进入正式白名单；默认只采元数据、链接、摘要、知识点标签 |
| B 试采白名单 | 70 <= 总分 < 85，且无一票否决项 | 小规模试采，人工抽检后再升级 |
| C 候选观察 | 55 <= 总分 < 70 | 暂不采集，只保留观察 |
| D 排除 | 总分 < 55，或触发一票否决 | 不进入白名单 |

## 一票否决项

出现任一项，直接标记为 D 排除。

1. 明显搬运他人课程、盗版教材、盗版网课、盗版讲义。
2. 主要目标是引流卖课、夸张提分、制造焦虑，教育内容占比低。
3. 存在低俗、暴力、歧视、阴谋论、伪科学、明显不适合儿童的内容。
4. 无法确认账号主体，或频繁更名、内容混杂严重。
5. 明确禁止任何自动化访问、采集、整理或二次使用。
6. 内容涉及实验、医疗、心理、安全等高风险主题，但缺乏资质或安全提醒。

## 量化评分

满分 100 分。

| 维度 | 分值 | 评分说明 |
|---|---:|---|
| 知识准确性 | 25 | 是否符合课标、教材和基础事实；是否有明显错误；是否能经第三方资料交叉验证 |
| 体系化程度 | 15 | 是否按年级、学科、章节、知识点持续输出；是否适合整理成知识库 |
| 儿童适宜性 | 15 | 语言、画面、价值观、难度、节奏是否适合小学生/初中生 |
| 来源可信度 | 15 | 官方、机构、实名教师、专家、学校/出版社账号优先；匿名账号降分 |
| 低营销程度 | 10 | 是否少广告、少引流、少焦虑营销；视频中教育内容占比是否高 |
| 更新稳定性 | 8 | 是否长期稳定更新；内容是否仍可访问 |
| 版权与合规可控性 | 8 | 是否能确认版权边界；是否适合只采链接、摘要和元数据 |
| 内容可结构化 | 4 | 是否容易抽取学科、年级、知识点、例题、摘要 |

## 抽样审核方法

每个账号先抽样 10 条公开视频，覆盖最近更新、热门视频、低播放视频、不同学科/主题。

每条视频记录：

```yaml
sample_id:
video_title:
video_url:
publish_date:
subject:
stage:
knowledge_points: []
accuracy_notes:
age_suitability_notes:
ad_or_sales_notes:
copyright_notes:
review_score_0_to_10:
reviewer:
review_date:
```

账号总分建议由两部分组成：

```text
账号总分 = 评分表总分 * 70% + 10 条样本平均分 * 10 * 30%
```

## 风险标签

| 标签 | 含义 |
|---|---|
| copyright_unknown | 版权边界不清晰 |
| marketing_heavy | 营销/卖课比例高 |
| too_advanced | 难度明显超出小学/初中 |
| mixed_content | 账号内容混杂，不全是教育内容 |
| safety_sensitive | 涉及实验、医疗、心理、安全等敏感主题 |
| platform_limited | 平台规则或 robots 限制较多 |
| needs_parent_review | 适合家长/教师先看，不适合儿童直接看 |
| official_source | 官方或机构来源 |
| teacher_verified | 实名教师或专家来源 |

## 合并规则

1. 同一账号如果你和 Codex 都收集到，保留分数更谨慎的一版作为初始分。
2. 只要任一方标记一票否决，需要人工复核后才能恢复。
3. A 类账号进入 `正式白名单.md`。
4. B 类账号进入 `试采白名单.md`。
5. C 类账号保留在 `候选视频博主白名单.md`。
6. D 类账号进入 `排除清单.md`，并记录排除原因，避免重复评估。
